## Default Permission

Default permissions for the plugin

#### This default permission set includes the following:

- `allow-ping`
- `allow-play`
- `allow-pause`
- `allow-resume`
- `allow-stop`
- `allow-next`
- `allow-previous`
- `allow-seek`
- `allow-get-state`
